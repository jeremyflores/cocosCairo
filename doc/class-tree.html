            • cocosCairo.EaseAction.EaseBounceIn: An AbstractEaseAction whose curve first bounces before reaching completion.
            • cocosCairo.EaseAction.EaseBounceInOut: An AbstractEaseAction whose curve first bounces before reaching completion, then again bounces as it reaches completion.
            • cocosCairo.EaseAction.EaseBounceOut: An AbstractEaseAction whose curve bounces as it reaches completion.
          • cocosCairo.EaseAction.AbstractEaseElastic: An AbstractEaseAction whose curve oscillates so as to produce an elastic effect.
            • cocosCairo.EaseAction.EaseElasticIn: An AbstractEaseAction which oscillates at the beginning before reaching completion.
            • cocosCairo.EaseAction.EaseElasticInOut: An AbstractEaseAction which oscillates both at the beginning and as it reaches completion.
            • cocosCairo.EaseAction.EaseElasticOut: An AbstractEaseAction which oscillates at the end as it reaches completion.
          • cocosCairo.EaseAction.AbstractEaseRateAction: An AbstractEaseAction whose curve is determined by a given rate.
            • cocosCairo.EaseAction.EaseIn: An AbstractEaseRateAction that starts slowly then increases speed as it nears completion.
            • cocosCairo.EaseAction.EaseInOut: An AbstractEaseRateAction that starts slowly, speeds up to being 50% complete, then slows back down as it nears completion.
            • cocosCairo.EaseAction.EaseOut: An AbstractEaseRateAction that starts quickly then decreases speed as it nears completion.
          • cocosCairo.EaseAction.EaseBackIn: An AbstractEaseAction whose curve first reverses before reaching completion.
          • cocosCairo.EaseAction.EaseBackInOut: An AbstractEaseAction whose curve first reverses then overshoots and corrects as it reaches completion.
          • cocosCairo.EaseAction.EaseBackOut: An AbstractEaseAction whose curve overshoots and then corrects as it reaches completion.
          • cocosCairo.EaseAction.EaseExponentialIn: An AbstractEaseAction that starts slowly then increases speed as it nears completion.
          • cocosCairo.EaseAction.EaseExponentialInOut: An AbstractEaseAction that starts slowly, speeds up to being 50% complete, then slows back down as it nears completion.
          • cocosCairo.EaseAction.EaseExponentialOut: An AbstractEaseAction that starts quickly then decreases speed as it nears completion.
          • cocosCairo.EaseAction.EaseSineIn: An AbstractEaseAction that starts slowly then increases speed as it nears completion.
          • cocosCairo.EaseAction.EaseSineInOut: An AbstractEaseAction that starts slowly, speeds up to being 50% complete, then slows back down as it nears completion.
          • cocosCairo.EaseAction.EaseSineOut: An AbstractEaseAction that starts quickly then decreases speed as it nears completion.
