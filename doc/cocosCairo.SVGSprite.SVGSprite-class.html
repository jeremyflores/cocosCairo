                                           SVGSprite

Renders SVG files to the screen. While Sprite can render SVG files, it first renders the SVG file to a pixel buffer, which forces it to be scaled by interpolating the pixels. SVGSprite, on the other hand, does allow scaling without pixel interpolation.

Note that if the SVG file is large, it is recommended to use Sprite as SVGSprite might slow down the application.
