setNode(self, node)

source code 

Sets the controller's Node. It will automatically register the view with the AbstractModel, if one has been set for this controller. Also, if there was previously a parented Node, the new Node will become a child of that parent and will inherit the same zOrder as its predecessor. By default, the parent is the Scene to which this Controller has been added.

Parameters:
  • node (Node) - The node.

setModel(self, model)

source code 

Sets the model for this controller. If the Node has been defined, it will unregister the Node from listening to the previous model (if it exists), then it will register the Node to listen to the new model. When the model changes, it will call Node.onModelChange.

Parameters:
  • model (AbstractModel) - The new model.
