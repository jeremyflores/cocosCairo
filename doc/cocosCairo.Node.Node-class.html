The base class for rendering items to the screen.

Note that the position is set relative to the Node's parent. If the Node's parent is the Director, the position is relative to the top-left corner of the screen (in general, this should only apply to Scenes).

setAnchorPoint(self, anchorPoint)

source code 

Sets how the Node is displayed on the screen relative to its position. E.g. Point(0,0) means that the Node will be drawn with its top left-most corner at the position, Point(1,1) means that the Node will be drawn with its bottom right-most corner at the position, and Point(0.5, 0.5) means that the Node will be drawn with its center at the position.

Parameters:
  • anchorPoint - Point, with both 0 <= x <= 1 and 0 <= y <= 1.
