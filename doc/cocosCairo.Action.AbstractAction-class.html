Class AbstractAction

The base class for all Actions. An Action is a convenience class used to perform some task over a period of time (or instantaneously). Actions usually perform some task on an object, which is referred to as the Action's "owner". The owner should typically be a Node.

Returns the owner of the Action when the Action is started via start. (When the Action is stopped via stop, it will no longer have an owner.) If the owner has not yet been defined, it will return None.

getOwner(self)

source code 

Returns the owner of the Action. If this is called while the Action is still running (that is, after start has been called but before stop has been called), it will return the owner which the Action is modifying. Otherwise, it will return None.

Returns: defined by subclasses (or None if the action is not running)
The owner of the Action.
