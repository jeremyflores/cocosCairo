setWrappingType(self, wrappingType)

source code 

This value is ignored if width is negative.

Set it to "none" if there should not be any wrapping, "word" if it should break only between words, "word-char" if it should try to break at words first before falling back to breaking between individual characters, or "char" if it should break between individual characters.

By default, wrappingType is "none".
