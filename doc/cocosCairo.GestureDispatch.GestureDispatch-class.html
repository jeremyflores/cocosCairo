Parameters:
  • widget (gtk.Widget) - The GTK widget in which the event occurred. This should normally be the GTKLayout.
  • event (gtk.gdk.MOTION_NOTIFY) - The GTK event with the event information.

_onMousePress(self, widget, event)

source code 

Private method. Receives mouse press events from GTK. It constructs a MouseGestureEvent, then dispatches it via dispatchMousePressGesture (or dispatchMouseDoublePressGesture if it was a double press). Note that, when a user double presses, two single-press events will be dispatched before the double-press event is dispatched.

Parameters:
  • widget (gtk.Widget) - The GTK widget in which the event occurred. This should normally be the GTKLayout.
  • event (gtk.gdk.BUTTON_PRESS) - The GTK event with the event information.
